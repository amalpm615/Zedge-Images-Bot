# Easy Way To Deploy

## Installation Instruction

- copy the PHP files to your webhost.
- Rename `config.sample.php` to `config.php`
- Edit `config.php`
- Now, go to `https://api.telegram.org/botTOKEN/setWebhook?url=URL/bot.php`
- Start the bot!
